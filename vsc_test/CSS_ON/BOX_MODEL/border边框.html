边框模型

不输入边框的宽和样式边框是无法出现的


允许简写,比如: Border: 1px soild red

而且不用按顺序，只需记住三要素,顺序一般这样写，方便更改和辨认

想要分开写表框可以加上某边的参数,比如:border-top: red 5px dotted
即使是单独设置某一条边，简写也必须写完三要素，不是简写就可以直接套用

表格示例
1.1 1.2 1.3
2.1 2.2 2.3
3.1 3.2 3.3

Tips:居中表格可以不写align，在css里写上margin:0 auto或者left,right都设置成auto

参数border-collapse: collapse可以让表格边框距离变为0
